# 抠图功能配置快速参考

## ✅ 配置已完成

Dockerfile.api 已经配置好所有必要的依赖，可以直接使用。

## 🚀 快速开始

```bash
# 1. 构建镜像
cd deploy
docker compose build api

# 2. 启动服务
docker compose up -d

# 3. 测试抠图功能
# 访问编辑器 → 上传图片 → 点击"抠图"按钮
# 首次使用会自动下载模型（约 10-30 秒）
```

## 🔧 环境变量配置

在 `docker-compose.yml` 中可以配置以下环境变量：

```yaml
services:
  api:
    environment:
      # Python 可执行文件路径（默认：python3）
      - AI_CUTOUT_PYTHON=python3
      
      # 抠图模型选择（默认：u2netp）
      # 可选值：u2netp（快速）、u2net（高质量）、isnet-general-use（最高质量）
      - AI_CUTOUT_REMBG_MODEL=u2netp
      
      # 抠图超时时间（毫秒，默认：60000）
      - AI_CUTOUT_WORKER_TIMEOUT=60000
      
      # 抠图提供商链（默认：rembg,aliyun-human-cutout,mock）
      # rembg: 本地模型抠图
      # aliyun-human-cutout: 阿里云人像抠图（需要配置 API Key）
      # mock: 演示模式（不实际抠图）
      - AI_CUTOUT_PROVIDER=rembg,mock
```

## 📊 模型对比

| 模型名称 | 大小 | 速度 | 质量 | 适用场景 |
|---------|------|------|------|---------|
| u2net | 176MB | ⚡⚡ 中等 | ⭐⭐⭐ 好 | **生产环境（默认）**、高质量要求 |
| u2netp | 5MB | ⚡⚡⚡ 快 | ⭐⭐ 中等 | 轻量级部署、快速预览 |
| isnet-general-use | 176MB | ⚡ 慢 | ⭐⭐⭐⭐ 很好 | 最高质量要求 |
| u2net_human_seg | 176MB | ⚡⚡ 中等 | ⭐⭐⭐ 好 | 人像专用 |

## 🎯 推荐配置

### 开发/测试环境
```yaml
environment:
  - AI_CUTOUT_REMBG_MODEL=u2netp  # 快速模型
  - AI_CUTOUT_PROVIDER=rembg,mock  # 本地模型 + 演示模式
  - AI_CUTOUT_WORKER_TIMEOUT_MS=30000  # 30秒超时
```

### 生产环境（小规模）
```yaml
environment:
  - AI_CUTOUT_REMBG_MODEL=u2net  # 高质量模型（默认）
  - AI_CUTOUT_PROVIDER=rembg,mock  # 本地模型 + 演示模式
  - AI_CUTOUT_WORKER_TIMEOUT_MS=60000  # 60秒超时
deploy:
  resources:
    limits:
      memory: 3G  # 增加内存
```

### 生产环境（大规模）
```yaml
environment:
  - AI_CUTOUT_PROVIDER=aliyun-human-cutout,rembg,mock  # 优先云服务
  - BAILIAN_API_KEY=your_api_key  # 阿里云 API Key
  - AI_CUTOUT_REMBG_MODEL=u2net  # 云服务失败时的备用方案
  - AI_CUTOUT_WORKER_TIMEOUT_MS=60000
```

## 🔍 常用命令

```bash
# 查看容器日志
docker logs poster_api --tail 100 -f

# 进入容器
docker exec -it poster_api bash

# 检查 Python 版本
docker exec poster_api python3 --version

# 检查 rembg 安装
docker exec poster_api pip3 list | grep rembg

# 手动下载模型
docker exec poster_api python3 -c "from rembg import new_session; new_session('u2netp')"

# 查看模型文件
docker exec poster_api ls -la /root/.u2net/

# 重启服务
docker compose restart api
```

## ⚠️ 注意事项

1. **首次使用**: 首次抠图会自动下载模型，需要 30-60 秒（u2net 模型 176MB）
2. **网络要求**: 模型下载需要访问 GitHub，如果网络受限请手动预下载
3. **内存要求**: 建议至少 2GB 可用内存，推荐 3GB
4. **并发限制**: CPU 版本建议限制并发数为 1-2，避免内存溢出
5. **模型缓存**: 模型下载后会缓存在 `/root/.u2net/` 目录，重启容器不会丢失
