## 📋 部署步骤

### 1. 解压部署包
```bash
unzip poster-design-docker-deploy.zip
cd poster-design/deploy
```

### 2. 配置环境变量
```bash
# Linux/Mac
cp env.template .env

# Windows
copy env.template .env
```

### 3. 编辑 .env 文件
需要配置以下内容：
- MySQL 数据库连接信息
- OAuth 统一登录中心配置
- 百炼 API 密钥
- 其他第三方服务配置

### 4. 初始化数据库
在远程 MySQL 数据库中执行 `mysql/init` 目录下的 SQL 文件。

### 5. 启动服务
```bash
docker compose -f docker-compose.yaml up -d --build
```
