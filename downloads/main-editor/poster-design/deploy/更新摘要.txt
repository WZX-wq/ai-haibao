========================================
  快速部署步骤
========================================

1. 解压部署包
   unzip poster-design-docker-deploy.zip
   cd poster-design/deploy

2. 配置环境变量
   cp env.template .env
   # 编辑 .env 填写数据库和 OAuth 配置

3. 初始化数据库
   # 在 MySQL 中执行 mysql/init 目录下的 SQL 文件

4. 启动服务
   docker compose -f docker-compose.yaml up -d --build

5. 测试抠图功能
   # 访问编辑器 → 上传图片 → 点击"抠图"
   # 首次使用会自动下载模型（30-60秒）

========================================
  重要提示
========================================

⚠️ 首次抠图会下载 u2net 模型（176MB）
⚠️ 需要至少 2GB 内存，推荐 3GB
⚠️ 需要访问 GitHub（下载模型）
⚠️ 如果内存不足，可切换到 u2netp 模型（5MB）
