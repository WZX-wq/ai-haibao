## 🚀 快速开始

### 1. 解压部署包
```bash
unzip poster-design-docker-deploy.zip
cd poster-design/deploy
```

### 2. 配置环境变量
```bash
cp env.template .env
# 编辑 .env 文件，填写数据库和 OAuth 配置
```

### 3. 初始化数据库
在 MySQL 中执行 `mysql/init` 目录下的 SQL 文件。

### 4. 构建并启动服务
```bash
docker compose -f docker-compose.yaml up -d --build
```

### 5. 测试抠图功能
1. 访问编辑器
2. 上传一张图片
3. 点击"抠图"按钮
4. 首次使用会自动下载模型（30-60 秒）
5. 后续抠图速度：5-15 秒/张

## ⚠️ 重要提示

### 1. 首次抠图
首次使用抠图功能时，系统会自动下载 u2net 模型（176MB）。请确保：
- 服务器可以访问 GitHub
- 有足够的磁盘空间（至少 500MB）
- 有足够的内存（至少 2GB）

### 2. 内存配置
如果服务器内存有限，可以在 `docker-compose.yaml` 中限制内存：
```yaml
services:
  api:
    deploy:
      resources:
        limits:
          memory: 3G
```

### 3. 模型切换
如果需要更快的速度（但质量略低），可以在 `.env` 中修改：
```bash
AI_CUTOUT_REMBG_MODEL=u2netp  # 轻量级模型，5MB
```

### 4. 网络问题
如果无法访问 GitHub，可以手动预下载模型：
```bash
docker exec -it poster_api bash
python3 -c "from rembg import new_session; new_session('u2net')"
exit
```
